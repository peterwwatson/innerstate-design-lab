# InnerState — explainer design candidates

Four competing full-page design directions for a single "explainer" screen of a
dark, meditative breathing-practice app — shared here purely to gather feedback on
the *design*. These are **candidates, not the product**: static mockups with placeholder
copy, no working app behind them. Open [`index.html`](index.html) to compare the four side
by side, then use the feedback prompt at the bottom of any page to tell me which one pulls
you in and where you stopped reading.
